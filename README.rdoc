## 💻  Sobre o projeto 
Serviço de API desenvolvido em Node.js que busca CEP,
O componente exporta uma função que retorna um endereço recebendo um cep como parâmetro


<h1 align="center">
<br>
  Como instalar 
</h1>

<p align="center">npm install </p>
<p align="center">npm start (executar serviço) </p>
<p align="center">npm test (testar aplicação) </p>

<hr />

<h1 align="center">
<br>
  Precisa ter Node.js instalado na maquina 
</h1>
